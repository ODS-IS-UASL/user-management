# ドローン航路システム ユーザ管理ライブラリ(Java版)

## 概要
- L3-identity-componentへのユーザ/事業者情報登録・更新・削除およびドローン航路のユーザ属性/事業者属性情報を提供する機能群です。
- ユーザ/事業者情報の登録・更新・削除、ユーザ属性/事業者属性情報の参照操作をすることができます。
